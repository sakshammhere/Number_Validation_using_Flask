About This Project

This project is a phone number validation tool that uses the Veriphone API to check the validity of phone numbers. It uses Flask for the backend and Bootstrap for the frontend. The project allows users to validate single phone numbers or upload a CSV file containing multiple phone numbers for batch validation. The results are displayed in a user-friendly format, showing whether each phone number is valid or not.

Purpose

The purpose of this project is to provide a simple and efficient way to validate phone numbers, ensuring that users can easily check the validity of their contacts. This can be particularly useful for businesses and individuals who need to maintain accurate contact information.

Key Features

  • Single phone number validation with real-time feedback.
  • Batch validation of phone numbers through CSV file upload.
  • User-friendly interface with Bootstrap styling.
  • Detects phone number validity, carrier, and country
  • Integration with the Veriphone API for accurate phone number validation.
  • Handles various phone number formats with country codes.
  • Storing validation results in a SQLite database for future reference.
  • Option to download a processed CSV with appended validation results

Technologies Used

  • Frontend : HTML, Bootstrap
  • Backend : Python(Flask)
  • Database : SQLite using SQLAlchemy
  • API : Veriphone API
  • Version Control : Git
